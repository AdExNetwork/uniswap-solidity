# Uniswap Solidity
Implementation of Uniswap contracts in Solidity with SafeERC20 and error reporting

## Installation:

1) Clone uniswap-solidity
```bash
$ git clone https://github.com/okwme/uniswap-solidity
$ cd uniswap-solidity
```

2) Install dependencies
```bash
$ yarn install
```

3) Run tests
```bash
$ yarn test
```
